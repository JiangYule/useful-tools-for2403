### 代码使用说明

#### 作业创建工具
1.  第三方库安装:
 docx keyboard
 `pip install docx`
 `pip install keyboard`
2.  更改配置:
 在settings.json文件中修改subjects的科目和path的文件路径
3. 一点小bug:
 如果word为空,本程序也会创建新页面并填写内容,暂未解决
